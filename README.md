# site-aro.studio

Aro Studio personal-brand SPA. React, TypeScript, Vite, React Router.

## Prerequisites

Node.js 18 or newer.

## Install

```bash
npm install
```

## Run

```bash
npm run dev
```

Dev server at http://localhost:5173

## Build

```bash
npm run build
```

## Preview

```bash
npm run preview
```

Serves the production build locally.

## Lint / format

- `npm run lint`
- `npm run format`
- `npm run format:check`
